# LAMP Stack Web Server Deployment with Automation

## Overview
Automated LAMP (Linux, Apache, MySQL, PHP) stack deployment using Bash shell scripts on Ubuntu Linux.

## Features
- Fully automated installation - no manual steps
- Reduced setup time by 70%
- Auto-configures Apache virtual host
- MySQL database setup with secure credentials
- PHP test page auto-deployed

## Tech Stack
- Linux (Ubuntu)
- Apache2
- MySQL
- PHP 8.x
- Bash Scripting

## How to Use

### Full LAMP Setup
```bash
chmod +x scripts/setup_lamp.sh
./scripts/setup_lamp.sh
```

### Database Setup
```bash
chmod +x scripts/setup_mysql.sh
./scripts/setup_mysql.sh
```

| Component | Version |
|-----------|---------|
